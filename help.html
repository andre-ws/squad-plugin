SQUAD

This tool searches through a point file to identify data anomalies that could indicate data quality issues. Each of these new columns will be created in the original site file. A "1" indicates that the anomaly exists. It is possible for a site to have multiple anomalies identified.

  • Anomaly_1: Missing coordinate.
  • Anomaly_2: Coordinate is truncated.
  • Anomaly_3: The point has an identical coordinate as another point in the file.
  • Anomaly_4: The point has an identical name as another point in the file.
  • Anomaly_5: The point is outside of the expected administrative unit, but is within two kilometers of the expected unit's border.
  • Anomaly_6: The point is outside of the expected administrative unit, and is greater than two kilometers of the expected unit's border.
